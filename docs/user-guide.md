# User Guide

## Getting Started

### 1. Viewing Public Availability
Visit `index.html` to:
- See all playgrounds and their availability status. Submit booking requests (no login required)

### Registered Users
1. Open `index.html`
2. Login with your credentials
3. Navigate using the sidebar

## Booking a Playground

1. Go to **Resources** or **New Booking**
2. Select a playground
3. Choose date and time slot (green = available)
4. Enter purpose/description
5. Click **Submit Booking Request**
6. Wait for admin approval

## Checking Booking Status

1. Go to **My Bookings**
2. Use tabs to filter: All, Pending, Approved, Rejected
3. Cancel pending bookings if needed

## For Faculty

### Upload Usage Record
After your booking is completed:
1. Go to **Upload Usage**
2. Select your booking
3. Upload photos/videos
4. Add remarks and any issues
5. Submit

## For Admins

### Approve/Reject Bookings
1. Go to **Approvals**
2. Review pending requests
3. Click **Approve** or **Reject**
4. Provide reason if rejecting

### View Reports
1. Go to **Reports**
2. Filter by date range or status
3. Export data if needed
